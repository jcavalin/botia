# BOTia
It's a discord bot made for fun and a tribute for someone that I really care.

### How to run?[^1]
```
npm start
```

### And in dev mode?[^1]
```
npm inspect
```

### How about the commands?
```
!b help
```

[^1]: You have to create a .env file with .env.example template or create the envs values yourself.
